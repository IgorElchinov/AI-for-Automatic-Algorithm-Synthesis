# Задача: распределение бюджета по каналам с убывающей отдачей

## Условие

Есть `n` рекламных каналов и общий бюджет `B`.  
Нужно распределить этот бюджет между каналами так, чтобы получить максимальный суммарный эффект.

Для каждого канала `i` выделяется целое неотрицательное число бюджетных единиц `x[i]`.

Эффект от вложения `x[i]` в канал `i` задаётся формулой:

\[
gain_i(x[i]) = a[i] \cdot \left(1 - e^{-b[i] \cdot x[i]}\right)
\]

где:
- `a[i]` — максимальный потенциальный эффект канала,
- `b[i]` — скорость насыщения канала,
- `x[i]` — выделенный этому каналу бюджет.

Суммарный эффект равен:

\[
\sum_{i=1}^{n} a[i] \cdot \left(1 - e^{-b[i] \cdot x[i]}\right)
\]

Требуется выбрать значения `x[0], x[1], ..., x[n-1]`, чтобы:

- `x[i] >= 0` для всех `i`,
- `x[i]` — целые числа,
- `x[0] + x[1] + ... + x[n-1] <= B`,

и при этом суммарный эффект был максимален.

---

## Формат входных данных

Во входном файле находятся:
- Натуральное число `n`
- список `a` длины `n`,
- список `b` длины `n`,
- целое число `B`.

### Ограничения
- `1 <= n <= 10`
- `0 <= B <= 10^4`
- `a[i] > 0`
- `b[i] > 0`

---

## Формат выходных данных

Нужно вывести список `x` длины `n`, где:
- `x[i]` — целое неотрицательное число,
- `sum(x) <= B`,
- суммарный эффект максимален.

Общение программы с тестирующей системой происходит через стандартные потоки ввода и вывода. Программа должна быть написана на языка Python 3.

---

## Пример

### Вход
4
120 90 150 110
0.18 0.25 0.12 0.20
20

### Выход
1 2 3 4
